With Häntä USB stick you can boot almost any PC to a secure Linux operating system variant called Tails.

Secure boot guarantees that there is no software traps that could compromise your privacy when using Häntä system.

It is pretty hard to attack to a Linux system. In Häntä it is still more difficult because the operating system is in read-only storage. You can trust that no program from the network can modify it. Every time you shut down the computer, you can be sure that it starts from the same state next time.

All your own data is in preconfigured cloud storage, not in stick or in local disk.

All the networking goes through Tor Internet routing where your traffic goes encrypted through several voluntary internet nodes so that the first one knows where you are and the last one knows what your message is. In between there are 1-2 additional nodes that guarantee that the first and last can not find each other.

In Häntä, there is internal firewall that guarantees that there is no other communication except Tor communication. You can bypass that but it is not recommended.

Speciality of Häntä system is that it comes with pre-configured unique identity.

Usually, it is hard to start using system: generate keys etc. Häntä comes with precreated identity.

You can log in with your identity to Häntä cloud services and start immediately sending encrypted email and chat with other people.

Häntä cloud also include 1 GB free storage in the cloud. Guaranteed for one year.
